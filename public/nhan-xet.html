XUẤT BẢN NHẬN XÉT HÀNG LOẠT

Hệ thống sẽ tự động đọc file Excel và xuất ra file ZIP chứa các bản nhận xét (Word).

Yêu cầu File Excel:

Dòng đầu tiên của file Excel bắt buộc phải có các cột tiêu đề sau (có phân biệt hoa/thường):

  • Số bản nhận xét (VD: 01-BNX/ĐT)
  • Tên chi bộ (VD: Chi bộ Sinh viên 1)
  • Thời gian (VD: ngày 15 tháng 05 năm 2026)
  • Tên Đoàn trường (VD: Đoàn Thanh niên Trường CNTT)
  • Họ và tên (VD: Nguyễn Văn A)
  • Nhận xét năng lực
  • Nhận xét khuyết điểm
Kéo thả file Excel vào đây hoặc click để chọn file
