HƯỚNG DẪN SỬ DỤNG HỆ THỐNG YUMOFFICE

Chào mừng đồng chí đến với tài liệu hướng dẫn sử dụng hệ thống quản lý văn bản tự động YUMOFFICE v2.0. Vui lòng đọc kỹ các quy tắc về định dạng file để hệ thống hoạt động chính xác 100%.

1. Quy trình tạo Bản nhận xét Đảng viên

Bước 1: Chuẩn bị file mẫu Word (.docx)

  • Mở file Word mẫu hành chính của Đoàn trường.
  • Tại những vị trí cần điền thông tin tự động, hãy sử dụng dấu ngoặc vuông kép: [[Tên_Biến]].
  • Ví dụ: Kính gửi đồng chí [[HoTen]], sinh viên lớp [[Lop]]...
  • ⚠️ Lưu ý cực kỳ quan trọng: Đôi khi Microsoft Word tự động kiểm tra lỗi chính tả (dấu gạch chân đỏ) và làm đứt gãy biến [[ ]] ngầm bên trong. Hãy đảm bảo gõ liền mạch hoặc Copy/Paste nguyên cụm biến vào Word để tránh lỗi phân mảnh XML.

Bước 2: Chuẩn bị file dữ liệu Excel (.xlsx)

  • File Excel cần có dòng tiêu đề cột rõ ràng.
  • Tên các tiêu đề cột trong Excel PHẢI chứa các từ khóa tương ứng với biến trong Word (Ví dụ: cột "Họ và Tên", cột "STT", cột "Điểm rèn luyện", "GPA").
  • Hệ thống được lập trình thuật toán làm sạch dữ liệu: tự động bỏ qua các dòng trống hoặc các dòng có đánh dấu "Đồng ý" lạc lõng. Dòng tiêu đề sẽ được hệ thống tự động dò tìm thông qua cột có chữ "Họ tên".

Bước 3: Xử lý và tải xuống

  • Truy cập mục Bản nhận xét chuyển Đảng chính thức.
  • Tải lên file Word và Excel đã chuẩn bị. Bấm XỬ LÝ & TẢI XUỐNG.
  • Hệ thống sẽ chạy ngầm, trộn dữ liệu và trả về cho đồng chí một tệp nén .ZIP. Trong đó chứa hàng loạt các file Word đã được điền tên từng người, xuất thành các file riêng lẻ vô cùng gọn gàng.

2. Tính năng Lịch sử tạo văn bản & Bảo mật dữ liệu

  • Không cần mạng Internet liên tục: Quá trình xử lý và trộn file Word diễn ra 100% bằng sức mạnh xử lý trên máy tính (trình duyệt) của đồng chí. Dữ liệu KHÔNG bị tải lên bất kỳ máy chủ nào.
  • Lưu trữ Ổ đĩa ảo (IndexedDB): Sau khi tạo file ZIP, hệ thống tự động lưu một bản sao an toàn vào ổ đĩa ảo của trình duyệt.
  • Đồng chí có thể sang tab LỊCH SỬ TẠO VĂN BẢN để tải lại file ZIP bất cứ lúc nào. Tuy nhiên, nếu đồng chí chủ động Xóa lịch sử duyệt web (Clear Cache) của trình duyệt, các file lưu trữ này sẽ bị mất đi.

3. Hỗ trợ Kỹ thuật

Nếu hệ thống báo lỗi "Không tìm thấy dòng tiêu đề" hoặc "Lỗi định dạng thẻ Word", vui lòng:

  1. Mở file Word ra, xóa đi gõ lại cụm [[HoTen]].
  2. Đảm bảo file Excel không bị gộp ô (Merge Cells) ở dòng tiêu đề cột.
  3. Liên hệ Ban Tổ chức - Kiểm tra Đoàn trường Đại học Bách khoa Hà Nội nếu cần hỗ trợ cấu trúc lại file mẫu.
